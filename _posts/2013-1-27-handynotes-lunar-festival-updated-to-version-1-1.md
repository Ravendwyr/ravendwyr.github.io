Version 1.1 of HandyNotes Lunar Festival was released on Curse a few minutes ago which should solve the startup issues experienced by... pretty much everyone who installed the addon.

The bug was caused by me using a simple date check to determine if the Lunar Festival was active in-game or not, but this turned out to cause a lot more problems than I intended to solve.

So I removed it.  The addon should now load properly for everyone.

An unfortunate side effect of this is that any unvisited Elders will now show up on the worldmap and the minimap all the time instead of only during the Lunar Festival.  I highly recommend that, if the marker pins are getting in the way, you disable or uninstall the addon when the Lunar Festival ends in early February.

This new version also fixes the marker pins of Elder Primestone and Elder Windtotem, with thanks going to GitHub user [killerpommes](https://github.com/killerpommes) for the patch.

You can download the new version of HandyNotes Lunar Festival from [CurseForge](https://www.curseforge.com/wow/addons/handynotes_lunarfestival).

I am very sorry for the inconveniences caused by this.  I tried to be clever, but it didn't work out this time.  It will not happen again.
